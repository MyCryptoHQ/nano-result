# nano-result

A miniature library for seperating concerns between expected and unexpected errors.

Inspired by Rusts' [Result enum](https://doc.rust-lang.org/std/result/enum.Result.html)

## Installation

### Using npm

`npm i nano-result`

### Using yarn

`yarn add nano-result`
